## Exercise B
This folder contains the resolution for the exercise B of the technical challenge given by Proofpoint.

### How can I run it?
1. Clone the GitHub repository and move to this folder
2. Open your favourite IDE, terminal or cmd.
3. Move the CSV file to be analyzed to this folder with the name of `books.csv`.
4. Execute the program in any of the following ways:
    * `py main.py` (Windows)
    * `python main.py`
    * `python3 main.py` (Linux)

### Output
- `report.md` where it is showed a summary of the most important metrics and the entire corrected book catalog.
