## Exercise C
This folder contains the resolution for the exercise C of the technical challenge given by Proofpoint.

### How can I run it?
1. Clone the GitHub repository and move to this folder
2. Open your favourite IDE, terminal or cmd.
3. Move the TXT file to be analysed to this folder with the name of `words.txt`.
4. Execute the program in any of the following ways:
    * `py main.py` (Windows)
    * `python main.py`
    * `python3 main.py` (Linux)

### Output
One file will be generated in this same folder:
- `report.md` where it is showed the 10 most repeated words found in the text. It considers words with accents (such as 'cómo' and 'como') as different words, because they can represent a different thing depending on the context.
